    Sort(String, Int32, IList<Int32>, IList<Int32>)

    Sorts in lexicographic order all the partial cyclic shifts (PCS) of the input of length 2 * pcsLength, given the position list of the PCS of the input of length pcsLength and their equivalence classes.

    Remarks

    The bootstrap of this process requires sorting PCS of length 1, i.e. single chars of the input. This can be done by any implementation of ICharsSorter, such as CountingSortCharsSorter, which sorts single chars of the input in linear time, in scenarios of small alphabets.
