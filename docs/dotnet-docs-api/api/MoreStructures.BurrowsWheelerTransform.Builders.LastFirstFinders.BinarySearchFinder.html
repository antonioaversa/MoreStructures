    Class BinarySearchFinder

    A NaiveFinder refinement which iterates over BWT and uses binary search on SortedBWT.

    Remarks

    Search over BWT has Time Complexity = O(n).

    Search over SortedBWT has Time Complexity O(log(n)).

    Space Complexity = O(1) for both search operations.
