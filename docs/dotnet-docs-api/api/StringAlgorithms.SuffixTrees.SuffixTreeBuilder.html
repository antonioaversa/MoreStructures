    BuildTree(TextWithTerminator)

    Build a ISuffixStructureEdge<TEdge, TNode, TPath, TBuilder> of the provided text, which is a n-ary search tree in which edges coming out of a node are substrings of text which identify edges shared by all paths to leaves, starting from the node.
